# BookHive 📚

BookHive is a full-stack digital library application designed to simplify book exploration and management. Built using React, Firebase, and Node.js, it offers a clean and responsive user interface with essential features such as authentication, real-time data integration, and seamless navigation.

---

## 🔍 Overview

BookHive provides users with a personalized and user-friendly experience to:

- Browse a collection of books
- Log in or sign up securely
- Filter and search books by type
- Add selected books to a cart
- Navigate smoothly across the application

This project demonstrates core frontend and backend development skills, UI design capabilities, and real-time database integration.

---

## 🧩 Tech Stack

| Technology     | Purpose                                      |
|----------------|----------------------------------------------|
| **ReactJS**     | Frontend development and component-based UI |
| **Firebase**    | Authentication and Firestore real-time database |
| **Node.js**     | Backend scripting for data upload and configuration |

- Setting up and integrating Firebase (Auth + Firestore)
- Writing scripts in Node.js to manage Firestore data
- Implementing reusable and modular React components
- Handling responsive design for multiple screen sizes

---

## ✨ Features

- 🔐 **User Authentication** – Sign-up and login via Firebase
- 📚 **Dynamic Dashboard** – Real-time book listing using Firestore
- 🔎 **Search and Filters** – Easily filter books based on type or keyword
- 🛒 **Cart System** – Add/remove books to a virtual cart
- 📱 **Mobile-Responsive UI** – Optimized for various screen sizes
